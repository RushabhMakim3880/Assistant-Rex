# 1.3 Installation & Setup Guide (Windows)

**Target Audience:** End Users, IT Admins  
**Platform:** Windows 10 / 11 (x64)  

---

## 1. Prerequisites
Before installing R.E.X., ensure your system meets the following requirements:
*   **OS:** Windows 10 Version 2004 or higher, or Windows 11.
*   **Processor:** Intel Core i5 / AMD Ryzen 5 or better.
*   **RAM:** 8 GB minimum (16 GB recommended).
*   **Microphone:** A dedicated USB microphone or high-quality headset is strongly recommended for best recognition.
*   **Internet:** Stable broadband connection (for AI processing).

## 2. Firewall & Antivirus Considerations
*   **Firewall:** R.E.X. uses a local Socket.IO server (TCP Port similar to 4000-5000 range). Ensure your firewall allows `python.exe` (backend) to listen on `localhost`.
*   **Antivirus:** Some heuristic scanners may flag unsigned installers. You may need to add an exclusion for:
    *   `%LOCALAPPDATA%\Programs\assistant-rex`
    *   `%APPDATA%\assistant-rex`

## 3. API Key Configuration
R.E.X. requires a Google Gemini API Key to function. This key is NOT bundled with the installer for security reasons.

1.  Obtain a key from [Google AI Studio](https://aistudio.google.com/).
2.  Search for "Edit the system environment variables" in Windows Start Menu.
3.  Click **Environment Variables...**.
4.  Under **User variables for [YourUser]**, click **New...**.
5.  **Variable name:** `GEMINI_API_KEY`
6.  **Variable value:** `your_api_key_string_here`
7.  Click **OK** on all dialogs.

## 3. Installation Steps

### Option A: Standard Installer (Recommended)
1.  Download `R.E.X Setup 1.1.0.exe` from the release distribution.
2.  Double-click the installer.
3.  **Note:** You may see a "Windows protected your PC" (SmartScreen) warning. This is expected for standard private releases.
    *   Click **More info**.
    *   Click **Run anyway**.
4.  The application will install to `%LOCALAPPDATA%\Programs\assistant-rex`.
5.  Launch "R.E.X" from the desktop shortcut or Start Menu.

### Option B: Portable Version
1.  Unzip the `win-unpacked` folder to a desired location (e.g., `C:\Apps\REX`).
2.  Run `R.E.X.exe` directly from the folder.

## 4. Post-Installation Verification
1.  On first launch, you should see the **Pulse Core** visualizer (Cyan circle).
2.  Say "REX" (or your configured wake word). The core should turn **Purple** (Thinking) or **Red** (Speaking).
3.  If the core remains static or no audio is heard, verify your Default Recording Device in Windows Sound Settings.

## 5. Troubleshooting
*   **App opens then closes immediately:**
    *   This usually indicates a missing API Key. Check the log file at `%APPDATA%\assistant-rex\main_process_log.txt`.
*   **"REX is not hearing me":**
    *   Ensure your microphone input volume is > 50% in Windows Settings.
    *   Check if another application (Zoom/Teams) has exclusive control of the microphone.
