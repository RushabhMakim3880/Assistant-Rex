# 3.1 Developer Setup Guide

**Repo:** `REX`  

---

## 1. Environment Requirements
*   **Node.js:** v18.0.0+
*   **Python:** v3.11 or v3.12
*   **Git:** Latest version
*   **OS:** Windows 10/11 (Required for audio/shell libraries)

## 2. Repository Structure
```
REX/
├── backend/            # Python Intelligence Engine
│   ├── server.py       # Socket.IO Entry & Tool Handler
│   ├── rex.py       # Audio Loop & Gemini Client
│   └── venv/           # Python Virtual Environment
├── electron/           # Desktop Container Code
│   └── main.js
├── src/                # React Frontend
│   ├── components/     # UI Widgets
│   └── App.jsx
├── release/            # Build Output
└── package.json        # Project Config
```

## 3. Installation Steps
1.  **Clone:** `git clone https://github.com/.../REX.git`
2.  **Frontend:**
    ```bash
    npm install
    ```
3.  **Backend:**
    ```bash
    cd backend
    python -m venv venv
    .\venv\Scripts\activate
    pip install -r requirements.txt
    ```

## 4. Running Locally (Debug Mode)
This command runs Vite (Frontend) and Electron (Container) + Python (Backend) concurrently with live-reload.
```bash
npm run dev
```

## 5. Building for Production
To create the `.exe` installer:
```bash
npm run build:win
```
*Note: Ensure `GEMINI_API_KEY` is set in your environment variables for local testing.*

## 6. Common Developer Issues
*   **"Missing ffmpeg":** Ensure `ffmpeg` is in your System Path or bundled in `backend/bin`.
*   **"Port 4000 in use":** Kill any orphan `server.py` processes.
*   **"Electron blank screen":** Check DevTools console for React compilation errors.
