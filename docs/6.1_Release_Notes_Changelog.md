# 6.1 Release Notes / Changelog

## Version 1.1.0 (Current)
**Release Date:** 2026-01-19

### New Features
*   **Pulse Core Visualizer:** Replaced heavy 3D particles with a lightweight, state-reactive 2D pulsing system (Cyan/Purple/Red).
*   **Audio Optimization:** Significantly improved microphone sensitivity for "Soft Voice" detection (VAD Threshold lowered).
*   **Installer:** Native Windows `.exe` installer now available.

### Breaking Changes
*   **Dependency:** This version requires Windows 10 Build 19041 or higher. Older builds are no longer supported.

### Bug Fixes
*   Fixed crash when `GEMINI_API_KEY` was missing (now logs error).
*   Fixed resource locking issues during build process.
