# 6.3 Uninstallation Guide

---

## 1. Removal Steps
1.  Open **Windows Settings** -> **Apps** -> **Installed Apps**.
2.  Search for **"R.E.X"**.
3.  Click the "..." menu and select **Uninstall**.
4.  Follow the prompts to remove the application binaries.

## 2. Residual Files Cleanup
The uninstaller removes code but preserves logs and user preferences. To completely wipe all data:
1.  Open File Explorer.
2.  Navigate to `%APPDATA%` (Enter `%APPDATA%` in the address bar).
3.  Delete the `assistant-rex` folder.
    *   *Warning: This deletes all logs and local settings.*
4.  Navigate to `%LOCALAPPDATA%\Programs`.
5.  Ensure the `assistant-rex` folder is gone.

## 4. Registry Entries
*   **Uninstall Key:** `HKEY_CURRENT_USER\Software\Microsoft\Windows\CurrentVersion\Uninstall\0682121e-12fa-55ba-87b6-cc31b0fc2398` (UUID varies by build).
*   **Preferences:** No other registry keys are written by the application.

## 5. Environment Cleanup
1.  Open "Edit the system environment variables".
2.  Remove `GEMINI_API_KEY` if you no longer need it for other applications.
